# 99疑难杂症

## 1、**ereturn clear**

"ereturn clear"是一个Stata命令，用于**清除最近一次命令的结果**。它通常用于在Stata程序中的命令之间清除返回结果。例如：

```
regress y x1 x2
ereturn clear
logit y x1 x2
```

在这个例子中，我们首先运行了一个回归命令，然后使用"ereturn clear"清除了该命令的返回结果。然后，我们运行了一个逻辑回归命令，而不会受到之前命令的返回结果的影响。

请注意，如果您在Stata命令窗口中键入"ereturn clear"，它将清除最近一次命令的返回结果。
