# 各种循环

```jsx
*保留最小值小于0的变量"new_*"

foreach var of varlist new* {
    qui summarize `var'
    local min = r(min)
    if `min' > 0 {
        drop `var'
    }
}
```

```jsx
foreach city of varlist 北京市-乌鲁木齐市 {
    gen treat_`city' = 自创区政策年 if city == "`city'"
}  //相当与first_year

foreach var of varlist treat_北京市-treat_乌鲁木齐市 {
	egen first_`var' = max(`var')
	}

foreach city of varlist 北京市-乌鲁木齐市 {
	gen 激活_`city'=`city'*(year>=first_treat_`city')
}	

*计数
forval i = 50(50)1000 {
    local j = `i' - 50
    gen count_`i' = 0
    foreach var of varlist 激活_北京市-激活_乌鲁木齐市 {
        replace count_`i' = count_`i' + (`j' < `var' & `var' <= `i')
    }
}
```

全国电信业务收入万元

全国移动电话年末用户数万户
乘以省_年末邮电局数处
省_年末电话机数部
省_年末邮电业务总量万元
省_年每百万人邮局数
省_年每百人固定电话数
省_每万人电话机数量
每一项，生成新的变量z*，写一个循环（双循环嵌套）

```jsx
local vars "全国互联网用户数 全国电信业务收入万元 全国固定电话年末用户数万户 全国移动电话年末用户数万户"
local factors "省_年末邮电局数处 省_年末电话机数部 省_年末邮电业务总量万元 省_年每百万人邮局数 省_年每百人固定电话数 省_每万人电话机数量"

foreach v of local vars {
    foreach f of local factors {
        gen z_`v'_`f' = `v' * `f'
    }
}
```

stata删除包含缺失值的变量，指定全部变量进行循环：

```jsx
foreach var of var _all {
    qui sum `var', meanonly
    if r(N) > 0  drop `var'
}
```
