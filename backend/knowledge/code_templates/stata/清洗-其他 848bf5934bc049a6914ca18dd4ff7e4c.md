将小于0变量的变量，替换为前一期

```jsx
foreach var of varlist new* {       //保留小于0的变量
    qui summarize `var'
    local min = r(min)
    if `min' > 0 {
        drop `var'
    }
}

sort city year
foreach var of varlist new* {
	bys city: replace `var' = l.`var' if `var' < 0
}
```

foreach var of var _all {

**检查householdID的长度，如果只有9位，则在末尾添加0**

householdID
0101041010
0101041010
0101041010
0101041010
0101041010
0101041010
010104101
010104101

```diff
* 检查householdID的长度，如果只有9位，则在末尾添加0

* 1. 生成一个新变量，用于存储修改后的householdID
gen new_householdID = householdID

* 2. 使用len()函数检查每个householdID的长度
* 如果长度为9，则在末尾添加0
replace new_householdID = householdID + "0" if length(householdID) == 9

* 展示结果
list householdID new_householdID, clean
```

## 列合并

```diff
gen 动态相对指标1 = string( 动态相对指标 ,"%9.2f") + "%"
```

| 动态相对指标 | 动态相对指标1 |
| --- | --- |
| 6.3684 | 6.37% |
| 7.1278 | 7.13% |
| 8.2386 | 8.24% |
| 9.7609 | 9.76% |
| 12.4212 | 12.42% |
| 14.5813 | 14.58% |
| 17.3788 | 17.38% |
| 21.0328 | 21.03% |
| 23.8551 | 23.86% |
1. 首先使用 `string()` 函数将原来的数值型变量 `dynamic_relative_index` 转换为字符型，其中 `"%9.2f"` 是指定转换格式，和之前介绍的 `format` 命令里的格式含义类似，确保数值以合适的格式（这里是保留两位小数等）进行转换，你可以根据实际需要调整这个格式说明符。
2. 然后通过字符串连接操作（使用 `+` 符号），在转换后的字符型数据后面添加 “%” 符号，并将结果存储在新生成的变量 `new_variable` 中。
