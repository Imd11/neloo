# 异质性分析

```diff

**异质性
use panel3, clear

xtreg h Int1 $z i.year   if Province=="北京市"|Province=="天津省"|Province=="河北省"|Province=="上海市"|Province=="江苏省"|Province=="浙江省"|Province=="福建省"|Province=="山东省"|Province=="广东省"|Province=="海南省"|Province=="辽宁省"|Province=="吉林省"|Province=="黑龙江" , fe r
est sto 异质性东1

xtreg h Int1 $z i.year  if Province=="山西省"|Province=="安徽省"|Province=="江西省"|Province=="河南省"|Province=="湖北省"|Province=="湖南省" , fe r
est sto 异质性中1

xtreg h Int1 $z i.year  if Province=="内蒙古自治区"|Province=="广西省"|Province=="重庆市"|Province=="四川省"|Province=="贵州省"|Province=="云南省"|Province=="陕西省"|Province=="甘肃省"|Province=="青海省"|Province=="宁夏省"|Province=="新疆维吾尔自治区"|Province=="西藏" , fe r
est sto 异质性西1

esttab 异质性* using 表-异质性分析-地区.rtf,  r2 obslast  nogaps drop (*year*) star(* 0.1 ** 0.05  *** 0.01) b(%6.4f) se(%6.4f)  r2(%6.4f)  compress replace

```

### 异质性分析中的聚类分析

```jsx
*经济发展水平异质性
use 测试malm, clear
collapse (mean) 人均地区生产总值元, by(城市) //面板数据各地区的平均值
cluster wardslinkage 人均地区生产总值元
cluster gen type1=group(3)
cluster dendrogram, cutnumber(13) ylabel(0(5000000000)20000000000, angle(horizontal))  yline(1190000000) scheme(s1mono)
keep id type1

save 聚类分析结果, replace
use 测试malm, clear
merge m:1 id using 聚类分析结果,nogen
xtreg y1 x $z  i.year if type1==3, fe r
est sto yreg_j1
xtreg y1 x $z  i.year if type1==2, fe r
est sto yreg_j2
xtreg y1 x $z  i.year if type1==1, fe r
est sto yreg_j3
esttab yreg_j* using 表-经济发展水平异质性.rtf,  r2 obslast  nogaps drop (*year*) star(* 0.1 ** 0.05  *** 0.01) b(%6.4f) se(%6.4f)  r2(%6.4f)  compress replace
```
