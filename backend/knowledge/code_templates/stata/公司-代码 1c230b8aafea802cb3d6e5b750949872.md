# 公司-代码

```jsx

***********************
use panel, clear
global z SOE Size Lev ROE Growth Board Dual

*基准回归
xtreg y x i.year, fe r
est sto xtreg1
xtreg y x $z i.year, fe r
est sto xtreg2
esttab xtreg* using 表-基准回归.rtf,  r2 obslast  nogaps drop (*year*) star(* 0.1 ** 0.05  *** 0.01) b(%6.4f) se(%6.4f)  r2(%6.4f)  compress replace

**稳健性检验
*替换被解释变量
use panel, clear
xtreg y x2 $z i.year, fe r
est sto wregt

*交互固定效应
use panel, clear
egen proid=group(Province)
reghdfe y x $z , absorb(i.proid#i.year code) 
est sto wreg交互固定效应

*分位数回归
sqreg y x $z , q(0.25,0.50,0.75)
est sto wreg分位数回归

esttab wreg* using 表-稳健性检验.rtf,  r2 obslast  nogaps drop (*year*) star(* 0.1 ** 0.05  *** 0.01) b(%6.4f) se(%6.4f)  r2(%6.4f)  compress replace

**内生性分析

*被解释变量前置一期
use panel, clear
sort id year
xtreg f.y x $z i.year, fe r
est sto nreg被解释变量前置一期
*控制变量滞后一期
use panel, clear
xtreg y x l.SOE l.Size l.Lev l.ROE l.Growth l.Board l.Dual i.year, fe r
est sto nreg控制变量滞后一期

esttab nreg* using 表-内生性分析.rtf,  r2 obslast  nogaps drop (*year*) star(* 0.1 ** 0.05  *** 0.01) b(%6.4f) se(%6.4f)  r2(%6.4f)  compress replace

**异质性分析
*地区异质性
use panel, clear
xtreg y x $z i.year if East==1 , fe r
est sto 地区异质性东1

xtreg y x $z  i.year if Mid==1, fe r
est sto 地区异质性中1

xtreg y x $z i.year if West==1 , fe r
est sto 地区异质性西1
esttab 地区异质性* using 表-地区异质性.rtf,  r2 obslast  nogaps drop (*year*) star(* 0.1 ** 0.05  *** 0.01) b(%6.4f) se(%6.4f)  r2(%6.4f)  compress replace

*要素投入异质性
use panel, clear
xtreg y x $z i.year if 劳动密集型==1 , fe r
est sto 要素投入异质性劳动密集型

xtreg y x $z i.year if 技术密集型==1 , fe r
est sto 要素投入异质性技术密集型

xtreg y x $z i.year if 资产密集型==1 , fe r
est sto 要素投入异质性资产密集型

esttab 要素投入异质性* using 表-要素投入异质性.rtf,  r2 obslast  nogaps drop (*year*) star(* 0.1 ** 0.05  *** 0.01) b(%6.4f) se(%6.4f)  r2(%6.4f)  compress replace

*中介效应
use panel, clear
xtreg m x $z i.year , fe r
est sto m1

xtreg y x m $z i.year , fe r
est sto m2
esttab m1 m2 using 表-中介.rtf,  r2 obslast  nogaps drop (*year*) star(* 0.1 ** 0.05  *** 0.01) b(%6.4f) se(%6.4f)  r2(%6.4f)  compress replace

*描述性统计
use panel, clear
logout, save(表-描述性统计) word replace: tabstat y x $z , by() statistics(mean sd min max count)

*相关性表
use panel, clear
estpost correlate y x $z ,matrix
esttab using 表-相关性及其显著性结果.rtf, unstack not noobs compress nogap replace star(* 0.1 ** 0.05 *** 0.01) b(%9.4f)

*相关性可视化
graph matrix y x $z ,scheme(s1mono) msize(tiny) mcolor(black)
```
