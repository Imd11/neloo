# 排名

```jsx
egen 排名 = rank(-总得分), unique
bysort 总得分: egen 总得分排名=min(排名) //分得分最高排名
sort 总得分
```

小方法：

```jsx
gsort -var
gen 排名=_n
```

对多变量循环进行：

```jsx
foreach v of varlist 总得分-h5得分{
	egen `v'_预排名=rank(-`v'), unique
	bysort `v' : egen `v'排名=min(`v'_预排名) //分得分最高排名
	drop `v'_预排名
}
```

```jsx
gsort -k l //数据按k降序，l升序排列 
```
