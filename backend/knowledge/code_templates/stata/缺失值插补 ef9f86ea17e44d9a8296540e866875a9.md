# 缺失值插补

按变量，用自身的线性趋势插补（适用于缺失值较少）：

```jsx
foreach var of varlist 住宅开发投资完成额（万元）全市（地区）- 规模以上工业企业数（个）全市（地区） {
	bys city: ipolate `var' year, gen(new_`var') epolate
}
```

按变量，用自身线性趋势，进行缺失值插补：

```jsx
*3.补缺缺失值
egen city=group(城市)
xtset city year
foreach var of varlist 住宅开发投资完成额（万元）全市（地区）- 公路货运量（万吨）全市（地区） {
    replace `var' = . if `var' == .
    qui xtreg `var' year , fe
    predict `var'_imp if `var' == .                              //新生成的"var_imp"为"var"缺失值的预测值。
	replace `var'=`var'_imp if `var' == .                        //将预测值插补到原变量"var"中。
}
drop *_imp
```

按城市、按变量，用该变量与地区生产总值（该变量无任何缺失）的线性关系进行的插补（适用缺失值较多）：

```jsx
foreach var of varlist 住宅开发投资完成额（万元）全市（地区）- 规模以上工业企业数（个）全市（地区） {
    replace `var' = . if `var' == .	
	  gen `var'_插补数据=.
	forval i =1/16 {
		 reg `var' gdp if city==`i'
		 predict `var'_imp if (`var' == .)&(city==`i')         //新生成的"var_imp"为"var"缺失值的预测值。
		 replace `var'_插补数据=`var'_imp  if city==`i'      //保留插补数据
		 replace `var'=`var'_插补数据 if (`var' == .)&(city==`i')    //将预测值插补到原变量"var"中。
		 drop `var'_imp
	}                                         
}
```
