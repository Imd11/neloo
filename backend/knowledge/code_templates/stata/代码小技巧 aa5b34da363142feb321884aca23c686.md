# 代码小技巧

**stata中，#delimit ;**

在Stata中，`#delimit ;` 是一个命令，用于改变命令的结束符。默认情况下，Stata的命令结束符是换行符，也就是说，每一行都被视为一个独立的命令。但是，如果你想要一个命令跨越多行，你可以使用 `#delimit ;` 命令来改变结束符为分号。这样，Stata就会把分号视为命令的结束，而不是换行符。

例如：

```
#delimit ;
gen y = x1 + x2
+ x3 + x4;

```

在这个例子中，`gen y = x1 + x2 + x3 + x4;` 是一个命令，尽管它跨越了两行。这个命令会在遇到分号时结束，而不是在换行时结束。
