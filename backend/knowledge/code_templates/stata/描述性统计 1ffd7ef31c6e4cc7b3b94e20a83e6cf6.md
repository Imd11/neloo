# 描述性统计

普通描述性分析结果的导出：

```jsx
*描述性统计
use data, clear
logout, save(表-描述性统计) word replace: tabstat density0 lngdp , by(treat) statistics(mean sd min max count)
```

```jsx
**描述性统计
use 数据, clear
keep TFP_LP 供应链金融水平 SOE Lev Growth Board Top1 FirmAge
outreg2 using 表-描述性统计结果.doc, replace sum(log) title(Decriptive statistics)
```
