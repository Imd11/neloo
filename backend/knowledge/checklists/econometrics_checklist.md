# 计量经济学分析检验清单

## 使用说明

在完成实证分析后，逐项检查以下内容。每个检查项标注了重要程度（★★★ 必须，★★ 重要，★ 建议）。

---

## 1. 数据质量检验 ★★★

### 1.1 缺失值检查
- [ ] 报告各变量缺失值比例
- [ ] 说明缺失值处理方式（删除/插补/不处理）
- [ ] 分析缺失是否随机（MAR/MCAR/MNAR）
- [ ] 缺失值删除后样本量是否足够

### 1.2 异常值检查
- [ ] 检查极端值（超出 3 倍 IQR 或 3 个标准差）
- [ ] 判断异常值是真实数据还是录入错误
- [ ] 报告异常值处理方式（Winsorize/删除/保留）
- [ ] 检验结果对异常值处理的敏感性

### 1.3 数据一致性
- [ ] 检查重复观测
- [ ] 检查 ID 唯一性
- [ ] 检查时间连续性（面板数据）
- [ ] 检查变量取值范围是否合理

---

## 2. 描述性统计 ★★★

### 2.1 基本统计量
- [ ] 报告均值、标准差、中位数、最大值、最小值
- [ ] 报告样本量
- [ ] 分组报告（处理组 vs 对照组）

### 2.2 分布检验
- [ ] 核心变量分布图（直方图/密度图）
- [ ] 偏度和峰度
- [ ] 正态性检验（如需要）

### 2.3 相关性检验
- [ ] 相关系数矩阵
- [ ] 检查多重共线性（VIF）
- [ ] 相关系数异常高的变量需特别关注

---

## 3. 因果识别 ★★★

### 3.1 DID 方法
- [ ] 平行趋势假设检验（图 + 统计检验）
- [ ] 事件研究法图示
- [ ] 政策前系数是否联合为零
- [ ] 排除预期效应

### 3.2 IV 方法
- [ ] 详细论证工具变量外生性
- [ ] 第一阶段 F 统计量 > 10
- [ ] 过度识别检验（如适用）
- [ ] 讨论 LATE 的政策含义

### 3.3 RDD 方法
- [ ] 断点处理变量连续性检验
- [ ] 其他协变量在断点处不跳跃
- [ ] 操纵检验（McCrary 检验）
- [ ] 带宽选择敏感性

### 3.4 匹配方法
- [ ] 匹配后平衡性检验
- [ ] 共同支撑假设检验
- [ ] 报告匹配后样本量变化

---

## 4. 回归诊断 ★★

### 4.1 模型设定
- [ ] 函数形式正确性（线性/对数/二次）
- [ ] 遗漏变量检验（Ramsey RESET）
- [ ] 交互项是否必要

### 4.2 标准误计算
- [ ] 说明聚类层级选择理由
- [ ] 聚类数量是否足够（一般 > 30-50）
- [ ] 对于小聚类数，考虑 Wild Bootstrap

### 4.3 残差诊断
- [ ] 异方差检验（Breusch-Pagan/White）
- [ ] 自相关检验（Durbin-Watson，面板数据）
- [ ] 残差正态性（大样本可放松）

---

## 5. 稳健性检验 ★★★

### 5.1 基本稳健性
- [ ] 逐步添加控制变量
- [ ] 改变样本期间
- [ ] 排除特殊观测（极端年份、特殊地区）

### 5.2 安慰剂检验
- [ ] 假政策时点检验
- [ ] 假处理组检验
- [ ] 假结果变量检验

### 5.3 替代性检验
- [ ] 替代因变量测度
- [ ] 替代自变量测度
- [ ] 替代估计方法

### 5.4 子样本分析
- [ ] 异质性分析（分组回归）
- [ ] 分位数回归（如适用）

---

## 6. 结果汇报 ★★

### 6.1 表格规范
- [ ] 系数显示到适当小数位
- [ ] 标准误在括号内
- [ ] 注明聚类层级
- [ ] 注明显著性水平标准
- [ ] 报告 N 和 R²

### 6.2 图表规范
- [ ] 图标题清晰
- [ ] 坐标轴标签完整
- [ ] 图例清晰
- [ ] 字体大小可读

### 6.3 文字说明
- [ ] 解释经济意义（不只是统计显著性）
- [ ] 讨论效应大小（effect size）
- [ ] 与现有文献对比
- [ ] 承认局限性

---

## 7. 常见错误警示 ★★★

### 绝对禁止
- [ ] 不得 p-hacking（选择性报告显著结果）
- [ ] 不得 HARKing（将探索性分析包装为验证性）
- [ ] 不得数据挖掘（没有理论基础的变量选择）

### 容易犯的错误
- [ ] 因果语言：只有可信因果识别策略才能说"影响/导致"
- [ ] 相关 ≠ 因果：描述性分析用"相关"、"关联"
- [ ] R² 不代表因果关系好坏
- [ ] 统计显著 ≠ 经济重要

### 格式错误
- [ ] 不要把对数写成 log 而不说明底数
- [ ] 不要混用 ln 和 log
- [ ] 百分点变化 vs 百分比变化要区分清楚

---

## 8. 最终检查 ★★★

### 提交前确认
- [ ] 所有代码可复现
- [ ] 数据与结果一致
- [ ] 表格与正文描述一致
- [ ] 参考文献格式正确
- [ ] 无隐私数据泄露

### 共同作者审查
- [ ] 至少一位合作者审核代码
- [ ] 结果数字双重检查
- [ ] 英文语法检查（如投英文期刊）

## 快速检验脚本

```python
def quick_data_check(df, key_vars):
    """快速数据质量检验"""
    print("=" * 50)
    print("数据质量快速检验")
    print("=" * 50)

    # 1. 基本信息
    print(f"\n样本量: {len(df)}")
    print(f"变量数: {len(df.columns)}")

    # 2. 缺失值
    print("\n缺失值比例:")
    for var in key_vars:
        miss_rate = df[var].isna().mean() * 100
        print(f"  {var}: {miss_rate:.2f}%")

    # 3. 描述统计
    print("\n描述统计:")
    print(df[key_vars].describe())

    # 4. 极端值
    print("\n极端值检测 (>3 std):")
    for var in key_vars:
        if df[var].dtype in ['float64', 'int64']:
            mean, std = df[var].mean(), df[var].std()
            n_extreme = ((df[var] - mean).abs() > 3 * std).sum()
            print(f"  {var}: {n_extreme} observations")

    # 5. VIF
    try:
        from statsmodels.stats.outliers_influence import variance_inflation_factor
        numeric_vars = [v for v in key_vars if df[v].dtype in ['float64', 'int64']]
        X = df[numeric_vars].dropna()
        if len(X) > 0:
            print("\nVIF (多重共线性):")
            for i, var in enumerate(numeric_vars):
                vif = variance_inflation_factor(X.values, i)
                print(f"  {var}: {vif:.2f}" + (" ⚠️" if vif > 10 else ""))
    except:
        pass

    print("\n" + "=" * 50)
```
